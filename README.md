# Booking Service (booking-go)

Веб-приложение для бронирования (отелей, билетов или услуг), разработанное на языке **Go (Golang)**. 
Основная цель проекта — **глубокое изучение и практическая отработка механизмов многопоточности (Concurrency) в Go** на примере высоконагруженного сервиса бронирования. 
Приложение эмулирует реальные бизнес-процессы (обработка заказов, проверка доступных мест, управление сессиями), где критически важно корректно и безопасно обрабатывать конкурентные запросы от множества пользователей одновременно.
---

## Суть проекта

Проект реализует полноценный сервис бронирования с клиент-серверной архитектурой. Он объединяет в себе бэкенд-логику на Go и фронтенд-часть для взаимодействия с пользователем. Сервис позволяет управлять доступными для бронирования объектами, обрабатывать заказы пользователей и сохранять состояние системы.

## 🛠️ Технологический стек

* **Бэкенд:** Go (Golang) — высокая производительность, строгая типизация и быстрая обработка запросов.
* **Фронтенд:** HTML / CSS — классический веб-интерфейс для отображения страниц бронирования и форм.
* **Инфраструктура:** Docker & Docker Compose — контейнеризация приложения и связанных сервисов (например, базы данных) для запуска одной командой.

---

## Архитектура и структура проекта

Проект структурирован в соответствии с лучшими практиками разработки на Go:

* `cmd/` — точка входа в приложение, инициализация конфигураций и запуск HTTP-сервера.
* `internal/` — внутренняя бизнес-логика приложения (обработчики запросов, сервисы, работа с базой данных).
* `static/` — статические файлы фронтенда (HTML-шаблоны, стили, скрипты) для рендеринга интерфейса.
* `docker-compose.yaml` — конфигурационный файл для мультиконтейнерного развертывания всей инфраструктуры проекта.
